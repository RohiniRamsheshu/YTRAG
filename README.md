# YT-RAG

YT-RAG is a Retrieval-Augmented Generation (RAG) project built using LangChain, vector databases, and LLM integrations.  
The project demonstrates how to load documents, generate embeddings, store vectors, and retrieve relevant context for intelligent question answering.

---

## Features

- Document loading using LangChain
- Text embedding generation using Sentence Transformers
- Vector storage with FAISS and ChromaDB
- Retrieval-Augmented Generation pipeline
- Integration with Groq and OpenAI models
- Notebook-based experimentation and prototyping

---
Tech Stack used :
Python
LangChain
LangGraph
Sentence Transformers
FAISS
ChromaDB
Groq API
OpenAI API
